Web Design Made Simple

What is LayoutCSS?

Verbose. Unopinionated. Plyable.

LayoutCSS was developed to have clear meaning and effect, to be flexible and not dictate what your website looks like or how to layout things, and to be used to fulfill your design needs as you see fit.

LayoutCSS is a modular, Sass-based CSS solution for a simple or moderate website design and layout. To exemplify, this very website was designed using nothing but core tools provided by LayoutCSS! Literally every style show was achieved using LayoutCSS.
